How Authentication Works

When you enter your Azure DevOps PAT or JIRA API token, it's stored only in your browser's localStorage. API calls are made directly from your browser to Microsoft/Atlassian servers using HTTPS. BugWizard acts purely as a client-side interface — no middleware, no proxies, no data interception.

Frequently Asked Questions

Everything you need to know

📱

What is Ozzy Device Farm?

Ozzy Device Farm is a cloud device lab built into BugWizard AI. It lets you test your apps on 50+ real Android & iOS devices in the cloud — with DevTools, screenshot capture, APK/IPA upload, and more. No physical devices needed.

❓

Does it work with self-hosted JIRA?

Yes! BugWizard AI supports both JIRA Cloud and self-hosted JIRA Server/Data Center. Just enter your instance URL and API token in settings.

🔐

Is my PAT/API token safe?

Absolutely. Your credentials are stored only in your browser's localStorage — never sent to any BugWizard server. All API calls go directly from your browser to Azure DevOps or JIRA.

🔑

Can I use my own AI API key?

Yes! BugWizard supports 10+ AI providers. Bring your own OpenAI, Claude, Gemini, Groq, or any other supported API key. You can also use free tiers from providers like Groq and OpenRouter.

💰

What's included in the free trial?

Everything. Full access to all features including AI Bug Creator, Retest Automator, Ozzy Device Farm, Ozzy AI, Video Compressor, API Tester, and all 10+ AI providers. No credit card required.

🌐

Does it work offline?

Partially. BugWizard is a PWA that caches core functionality. You can draft bugs offline, but pushing to Azure DevOps/JIRA and using cloud devices requires an internet connection.
